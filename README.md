This repository contains the design and simulation files for a scalable and cost-effective campus backbone network for the **University of Moratuwa**, together with the **existing ENTC LAN model** and its proposed backbone integration. The project was developed for **EN2150 – Communication Network Engineering** and uses **Cisco Packet Tracer** to validate connectivity, routing behavior, and overall network performance. :contentReference[oaicite:0]{index=0}

The proposed university network follows a **hybrid topology** consisting of a **partially meshed ring backbone** with **star-based local area networks** inside buildings. This design was selected to balance **scalability, redundancy, cabling cost, and performance** while supporting future campus expansion. :contentReference[oaicite:1]{index=1}

The repository includes:
- Packet Tracer simulation files for the **ENTC LAN** and the **UoM backbone**
- IPv4 and IPv6 addressing plans
- VLAN-based segmentation for the university backbone
- OSPF-based routing for inter-network communication
- Simulated services such as **DNS, web, and email**
- Report figures and supporting network design assets

For ENTC, the LAN model is based on the observed building network structure, including **floor-level access switches, an ENTC core switch, fiber uplinks, wireless access points, MDF/IDF-style organization, and structured copper/fiber cabling concepts**.
